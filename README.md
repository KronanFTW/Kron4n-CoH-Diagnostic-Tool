# Kron4n CoH Diagnostic Tool

**Kron4n CoH Diagnostic Tool** is a community-made read-only diagnostic helper for **Company of Heroes** and related mods.

It is made to help users collect useful troubleshooting information when Company of Heroes or a mod does not start correctly.

## Important Notice

This tool is **NOT** a Python executable package.  
This tool is **NOT** a game executable package.  
This tool is **NOT** modified, supervised, or maintained by third-party Steam users.  
This tool does **NOT** launch the game.  
This tool does **NOT** download game files.  
This tool does **NOT** modify Company of Heroes or mod files.

The tool only checks local Company of Heroes and mod installation paths and creates a diagnostic log for troubleshooting.

## What the Tool Does

The diagnostic tool checks common Company of Heroes installation paths, mod folders, launcher-related files, and configuration/log locations.

It then creates a troubleshooting report named:

```text
Kron4n's_diagnostic_log.txt
```

This log can be shared when asking for support.

## What the Tool Does Not Do

The tool does not install mods.  
The tool does not repair game files.  
The tool does not bypass Steam, Relic, SEGA, or any game protection.  
The tool does not include game executables or mod executables.  
The tool does not collect passwords, accounts, private browser data, or unrelated personal files.
